   - 一括データ更新

2. [🔔 Slack メニュー](./sequence-slack.md)
   - Slack手動送信
   - Slackフィルタ設定

   - 定期実行のSlack通知

3. [🔍 tool メニュー](./sequence-tool.md)
   - チケット詳細ビューア
